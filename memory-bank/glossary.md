# Glossary — Noa

## Dernière mise à jour
2026-04-14

## Termes projet

| Terme | Définition |
|-------|-----------|
| **Noa** | Application de gestion de dossiers CIR (Crédit Impôt Recherche) |
| **Dossier** | Ensemble organisé de sections représentant un projet de R&D |
| **Section** | Module de contenu dans un dossier CIR (ex: presenting, technical, financial) |
| **Tile** | Composant UI principal représentant une section dans l'interface |
| **TilePanel** | Panneau latéral affichant le contenu détaillé d'une tile sélectionnée |
| **CIR** | Crédit Impôt Recherche — dispositif fiscal français pour la R&D |
| **Dépense CIR** | Poste de dépense éligible au CIR (personnel, amortissements, sous-traitance) |

## Termes techniques

| Terme | Définition |
|-------|-----------|
| **Composable** | Fonction réutilisable Vue 3 utilisant l'API Composition API (équivalent React hooks) |
| **Nuxt plugin** | Module Nuxt auto-enregistré au démarrage de l'application |
| **Appwrite** | BaaS (Backend as a Service) utilisé pour l'authentification |
| **Supabase** | Plateforme PostgreSQL utilisée pour la base de données principale |
| **Sentry** | Service de monitoring d'erreurs et de performance |
| **Memory Bank** | Système de mémoire structurée du projet pour les agents IA |
| **Vibe Coding** | Approche de développement assistée par IA avec conventions structurées |
